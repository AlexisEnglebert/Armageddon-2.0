# Armageddon 2.0
Armageddon 2.0 is a free 3D/2D game engine under DirectX11.

![Alt text](https://github.com/Alexisloic21/Armageddon-2.0/blob/main/Preview/BloomPreview.png?raw=true)

## Features
* PBR
* Cascade shadow map
* Physically based volumetric fog (WIP)
